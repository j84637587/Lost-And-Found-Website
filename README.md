# Lost-And-Found-Website
此專案目的，開發一網站，提供學校學生做失物照領集中管理之網站。

本專案使用PHP原生開發、前端使用bootstrap美化、MYSQL做資料庫管理。

# 安裝需求
- PHP >= 7.2 
- MYSQL >= 5.7
- Apache
